IP/INVENTORY // CONTROL PANEL
● OFFLINE
Operations
Query

Add IP Addresses to Pool

Insert one or more IPv4 or IPv6 addresses into the inventory. Duplicates are ignored.

POST /ip-inventory/ip-pool

    

  


  
  

    
Reserve IP Address

    
Reserve a free IP for a serviceId. Reserved IPs are auto-released if not assigned within the timeout window.


    

      
POST /ip-inventory/reserve-ip


      

        
        
      

      

        
        
      


      
    


    

      

        
        
      

      

    

  


  
  

    
Assign IP to Service

    
Assigns listed IP addresses to the given serviceId (transitions free/reserved → assigned).


    

      
POST /ip-inventory/assign-ip-serviceId


      

        
        
      

      

        
        

          

            
            
          

        

        
      


      
    


    

      

        
        
      

      

    

  


  
  

    
Terminate IP Assignment

    
Releases listed IPs from the serviceId, returning them to free status.


    

      
POST /ip-inventory/terminate-ip-serviceId


      

        
        
      

      

        
        

          

            
            
          

        

        
      


      
    


    

      

        
        
      

      

    

  


  
  

    
Change Service ID

    
Re-assigns all assigned IPs from one serviceId to another. Used for service migrations.


    

      
POST /ip-inventory/serviceId-change

Returns all assigned IP addresses for a given serviceId.


    

      
GET /ip-inventory/serviceId?serviceId=…
